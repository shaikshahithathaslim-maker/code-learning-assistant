🎯 Lesson 1: Advanced Functions

Learn about higher-order functions, closures, and functional programming concepts in JavaScript.

// Higher-order function example function createMultiplier(factor) { return function(number) { return number * factor; }; } const double = createMultiplier(2); console.log(double(5)); // Output: 10

🧠 Code Explanation:

This demonstrates a closure where the inner function remembers the 'factor' variable from its outer scope, even after createMultiplier has finished executing.
